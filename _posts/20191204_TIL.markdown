## 12/04 수요일



1. 알고리즘
   - 점근적 표기법(**Θ**, **Ω**,**O**) 을 학습하였다.
     - **Θ** 표기법은 알고리즘 성능이 특정 범위안에 존재할 때 사용한다.
     - **Ω** 표기법은 알고리즘 성능이 특정 함수보다 같거나 나쁘다 (점근적 하한선)
     - **O** 표기법은 알고리즘 성능이 특정 함수보다 같거나 좋다. (점근적 상한선)
       - 최악의 경우를 표현 할때 주로 사용한다.
